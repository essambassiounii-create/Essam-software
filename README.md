# Essam Software - موقع شركة البرمجيات

موقع احترافي لعرض وتسويق البرامج والتطبيقات.

## 🚀 التشغيل

```bash
python manage.py runserver 0.0.0.0:5000
```

## 🔑 لوحة التحكم

- الرابط: `/admin/`
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

## 📄 الصفحات

- `/` — الصفحة الرئيسية
- `/about/` — من نحن
- `/products/` — البرامج
- `/services/` — الخدمات
- `/contact/` — تواصل معنا
- `/admin/` — لوحة التحكم

## ⚙️ التقنيات

- **Backend:** Django 5.2
- **Frontend:** Bootstrap 5 RTL + CSS مخصص
- **Database:** SQLite (تطوير) / PostgreSQL (إنتاج)
- **Admin:** Django Jazzmin
